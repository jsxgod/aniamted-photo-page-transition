## Run Locally

Clone the project files or download them manually

```bash
  git clone https://github.com/jsxgod/model-agency.git
```

Go to the project directory

```bash
  cd model-agency
```

Install dependencies

```bash
  npm install
```

Start the application

```bash
  npm run start
```
